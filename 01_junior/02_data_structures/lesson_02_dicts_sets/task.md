## Завдання 2 — Підрахунок слів
Напиши функцію `word_count(text: str) -> dict[str, int]` яка підраховує кількість кожного слова в тексті (без урахування регістру, без пунктуації).
```python
word_count("the quick fox the fox") → {"the": 2, "quick": 1, "fox": 2}
```


1. Хто є в обох класах одночасно
2. Хто є тільки в math_class

4. Хто є тільки в одному класі (але не в обох)

## Завдання 4 — Дедублікація
Напиши функцію `deduplicate(lst: list) -> list` що прибирає дублікати **зі збереженням порядку**.
```python
deduplicate([3, 1, 4, 1, 5, 9, 2, 6, 5]) → [3, 1, 4, 5, 9, 2, 6]
```

## Завдання 5 (Challenge) — Групування
Напиши функцію `group_by(items: list[dict], key: str) -> dict[str, list]`
яка групує список словників за значенням поля.
```python
users = [{"name": "A", "city": "Kyiv"}, {"name": "B", "city": "Lviv"}, {"name": "C", "city": "Kyiv"}]
group_by(users, "city") → {"Kyiv": [{"name":"A",...}, {"name":"C",...}], "Lviv": [{"name":"B",...}]}
```
