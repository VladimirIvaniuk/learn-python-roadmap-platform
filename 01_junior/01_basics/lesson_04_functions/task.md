# Завдання — Урок 4: Функції

## Завдання 1 — Базові функції
Напиши три функції з анотаціями типів і docstring:
- `square(n: int) -> int` — квадрат числа
- `is_even(n: int) -> bool` — перевірка парності
- `max_of_two(a, b)` — максимум з двох чисел (без вбудованого `max`)

---

## Завдання 2 — Default arguments
Напиши функцію `greet(name: str, greeting: str = "Привіт", times: int = 1) -> str`
яка повертає рядок привітання `times` разів.
`greet("Іван")` → `"Привіт, Іван!"`
`greet("Іван", times=3)` → `"Привіт, Іван! Привіт, Іван! Привіт, Іван!"`

---

## Завдання 3 — *args та **kwargs
Напиши функцію `stats(*numbers)` яка приймає довільну кількість чисел і повертає словник:
```python
{"min": ..., "max": ..., "sum": ..., "avg": ..., "count": ...}
```

---

## Завдання 4 — Замикання (Closure)
Напиши функцію `make_multiplier(factor: int)` яка повертає функцію,
що множить аргумент на `factor`.
```python
double = make_multiplier(2)
triple = make_multiplier(3)
print(double(5))   # 10
print(triple(5))   # 15
```

## Завдання 5 (Challenge) — Декоратор логування
Напиши декоратор `@log_call` який перед кожним викликом функції виводить:
`Виклик: <ім'я_функції>(<аргументи>)` та після: `Результат: <результат>`

Використай `functools.wraps` щоб зберегти метадані функції.
