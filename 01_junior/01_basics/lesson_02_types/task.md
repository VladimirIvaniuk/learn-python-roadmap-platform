# Завдання — Урок 2: Типи даних та операції

## Завдання 1 — Арифметика
Дано `a = 17`, `b = 5`.
Виведи результати усіх 7 операцій: `+`, `-`, `*`, `/`, `//`, `%`, `**`.
Формат: `17 // 5 = 3`

---

## Завдання 2 — Рядкові методи
Дано рядок: `text = "  hello, world!  "`
Виведи:
1. Рядок без пробілів по краях
2. Перша літера велика (решта маленькі) — `capitalize()`
3. Скільки разів зустрічається буква `l`
4. Замінити `world` на `Python`
5. Розбити за комою — отримай список з 2 елементів

---

## Завдання 3 — Конвертація типів
Напиши функцію `safe_int(value)` яка:
- повертає `int(value)` якщо конвертація можлива
- повертає `0` якщо ні (без виключення)

Перевір: `safe_int("42")` → 42, `safe_int("abc")` → 0, `safe_int(3.7)` → 3

---

## Завдання 4 — Truthiness
Для кожного значення виведи `"truthy"` або `"falsy"`:
`0`, `1`, `""`, `"hello"`, `[]`, `[0]`, `None`, `False`, `True`, `0.0`, `0.001`

Підказка: `print("truthy" if value else "falsy")`

## Завдання 5 (Challenge) — Калькулятор
Запитай у користувача два числа і операцію (`+`, `-`, `*`, `/`).
Виконай відповідну операцію і виведи результат.
Обробляй: ділення на нуль, невідому операцію, нечислове введення.
Виведи результат з 2 знаками після коми.

> ⚠️ Для веб-ранера платформи:
> якщо `input()` дає `EOFError`, зроби platform-варіант із фіксованими значеннями.
>
> **Platform-версія:**
> ```python
> a_str = "10"
> op = "/"
> b_str = "3"
> ```
>
> **Local-версія:**
> ```python
> a_str = input("Перше число: ")
> op = input("Операція (+, -, *, /): ")
> b_str = input("Друге число: ")
> ```
