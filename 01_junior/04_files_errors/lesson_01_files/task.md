# Завдання — Урок 1: Файли та серіалізація

## Завдання 1 — Читання та запис
1. Запиши у файл `words.txt` 10 слів (кожне на новому рядку)
2. Прочитай файл і виведи слова в алфавітному порядку
3. Порахуй загальну кількість символів у всіх словах
4. Знайди та виведи найдовше слово

---

## Завдання 2 — CSV
Створи CSV-файл `students.csv` з колонками `name,age,grade`.
Додай 5 студентів. Потім напиши функцію що:
1. Читає CSV і повертає список словників
2. Фільтрує студентів з оцінкою ≥ 80
3. Записує відфільтрованих в `top_students.csv`

---

## Завдання 3 — JSON конфіг
Напиши функцію `load_config(path: str, defaults: dict) -> dict` яка:
- Завантажує JSON-конфіг якщо файл існує
- Повертає `defaults` якщо файл не існує
- Зберігає defaults у файл якщо його не було
- Мержить defaults із завантаженим (defaults перекриваються файлом)

---

## Завдання 4 — Логгер у файл
Напиши клас `FileLogger(filename)` з методами:
- `log(level, message)` — записує `[2026-03-01 12:00:00] LEVEL: message`
- `info(msg)`, `warning(msg)`, `error(msg)` — зручні методи
- `get_errors()` — повертає список рядків з помилками (рівень ERROR)

---

## Завдання 5 (Challenge) — pathlib обхід директорій
Напиши функцію `analyze_directory(path: str) -> dict` яка через `pathlib`:
1. Рекурсивно обходить директорію
2. Підраховує файли за розширенням (`{".py": 5, ".md": 3, ...}`)
3. Знаходить найбільший файл
4. Повертає загальний розмір усіх файлів
